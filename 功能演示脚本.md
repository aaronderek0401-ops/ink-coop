## 场景 1: 启动服务（10秒）

**操作**:
1. 打开 PowerShell
2. 输入: `cd G:\A_BL_Project\inkScree_fuben`
3. 输入: `python tools/ttf_to_gfx_webservice.py`
4. 等待服务启动成功

**旁白**:
"首先，我们启动 Python Web 服务。运行这个命令后，服务会在本地 5000 端口启动，提供 6 个 API 端点。"

## 场景 5: 代码集成（30秒）

**操作**:
1. 在 VS Code 中打开 ink_screen.cpp
2. 在文件顶部添加 include（Ctrl+V 粘贴）
3. 滚动到显示函数
4. 添加 drawBitmap 代码

**旁白**:
"最后，我们在代码中使用刚才导入的位图。首先粘贴 include 语句，然后在显示函数中调用 drawBitmap。"

**屏幕显示**:
```cpp
// ink_screen.cpp

#include "../fonts/fangsong16pt_bitmaps.h"
#include "../bitmap/logo_bitmap.h"

void displayContent() {
    // 显示字体
    drawFangsongString(display, 10, 50, "你好世界");
    
    // 显示图片
    display.drawBitmap(10, 100, IMAGE_BITMAP, 
                      IMAGE_WIDTH, IMAGE_HEIGHT, 
                      GxEPD_BLACK);
}
```

**操作（续）**:
5. 保存文件
6. 按 Ctrl+Shift+P
7. 选择 "ESP-IDF: Build your project"
8. 等待编译完成
9. 选择 "ESP-IDF: Flash your project"
10. 等待上传完成

**旁白**:
"保存文件后，编译并上传到 ESP32。完成后，墨水屏上就会显示我们的文字和图片了。"

**屏幕显示**:

## 拍摄提示

1. **屏幕录制**: 使用 OBS 或 Windows 游戏栏录制
2. **鼠标高亮**: 启用鼠标点击效果
3. **画质**: 至少 1080p
4. **音频**: 清晰的旁白或背景音乐
5. **字幕**: 添加关键操作的文字说明
6. **过渡**: 场景切换使用淡入淡出
7. **特写**: 重要信息放大显示
