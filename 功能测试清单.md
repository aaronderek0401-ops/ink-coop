# 功能测试清单

## ✅ 测试步骤

### 1. 启动服务
- [ ] 运行 `python tools/ttf_to_gfx_webservice.py`
- [ ] 确认看到 6 个 API 端点信息
- [ ] 确认服务运行在 http://localhost:5000

### 2. 测试字体库管理
- [ ] 打开 web_layout.html
- [ ] 在 TTF 字体管理部分找到"📚 字体库管理"
- [ ] 点击"🔍 查询字体库"按钮
- [ ] 确认能看到 fonts 文件夹下的字体文件列表
- [ ] 点击某个字体的"导入"按钮
- [ ] 确认弹出确认对话框
- [ ] 确认 include 路径已复制到剪贴板

### 3. 测试图片转换和保存
- [ ] 在"📷 图片转位图"部分上传一张图片
- [ ] 调整宽度和高度
- [ ] 点击"预览处理效果"
- [ ] 确认能看到处理后的黑白效果
- [ ] 点击"转换为位图数组"
- [ ] 确认转换成功，显示字节数
- [ ] 点击"💾 保存到服务器"按钮
- [ ] 确认显示保存成功和路径
- [ ] 检查 components/bitmap/ 文件夹是否有新文件

### 4. 测试图片库管理
- [ ] 在"🖼️ 图片库管理"部分
- [ ] 点击"🔍 查询图片库"按钮
- [ ] 确认能看到 bitmap 文件夹下的位图文件列表
- [ ] 确认显示文件名、尺寸、大小
- [ ] 点击某个位图的"预览"按钮
- [ ] 确认能看到位图的黑白预览
- [ ] 确认显示位图详细信息
- [ ] 点击"导入"按钮
- [ ] 确认弹出确认对话框
- [ ] 确认 include 路径和使用示例已复制

### 5. 测试错误处理
- [ ] 停止 Python 服务
- [ ] 点击"查询字体库"，确认显示错误提示
- [ ] 点击"查询图片库"，确认显示错误提示
- [ ] 点击"保存到服务器"，确认显示错误提示
- [ ] 重启服务，确认功能恢复正常
