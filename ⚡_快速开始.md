## 🔧 或者手动启动

### 步骤 1：启动服务器

打开命令行或 PowerShell：
```bash
cd g:\A_BL_Project\inkScree_fuben
python config_server.py
```

等待看到：
```
✓ Running on http://127.0.0.1:5001
```

### 步骤 2：打开网页

直接在文件管理器中双击：
```

```

或在浏览器地址栏输入：

file:///G:/A_BL_Project/inkScree_fuben/web_layout_standalone.html

### 步骤 3：验证成功

应该看到：
- 🟢 **绿色状态条** → PC 模式已连接

## 💡 常见问题

**Q: 为什么显示蓝色状态？**
A: Flask 服务器没有运行。执行 `python config_server.py`

**Q: 图标还是加载不了？**
A: 运行测试脚本 `python 测试PC端Icon加载.py` 诊断问题

**Q: 可以从其他电脑访问吗？**
A: 可以！使用 `http://你的IP:5001` 替代 `localhost:5001`
